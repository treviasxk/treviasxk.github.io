Theme

Modify the variables in the CSS file at /data/css/layout.css to edit your blog's theme.

The theme section explains how to change the appearance of your blog. By editing the CSS file, you can adjust colors, fonts, and layout to match your desired style.

Supabase

Modify the variables in the js file at /data/js/config.js to connect to the API.

This section is about connecting your blog to the Supabase API. Supabase is a backend-as-a-service that provides a database and authentication. You need to configure the JavaScript file to ensure your blog can communicate with the Supabase backend.

Configuration

To configure your DevBlog, edit the JavaScript file at /data/js/config.js. This file contains settings for connecting to your database, API keys, and other configuration options necessary for your blog to function correctly. Make sure to replace placeholder values with your actual configuration details.
